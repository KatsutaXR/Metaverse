## 概要
VR対応のメタバース。  
VRChatによくあるカメラやペン、動画プレイヤーやアバター機能などを実装しました。  
また、基本的に各機能は同期するように実装しています。  
現状はMetaQuestでのみ動作確認済みです。  

画像！！！！！

## 使用技術
- Unity 6
- XR Interaction Toolkit
- Photon Fusion 2
  - Fusion Physics
- Photon Voice 2
- VContainer
- UniRx
- UniTask
- Visual Studio Code

## 主な機能
- VRインタラクション(移動、掴む操作、UI操作等)
- セッションの作成、参加
- 動画プレイヤー
- ワールド上で線を描画するペン
- 写真撮影(カメラ)
- セッション内情報表示UI
- ライティングのプリセット変更
- ボイスチャット
- アバター変更
- プロフィール設定と閲覧
- HMDの基準地点リセット(位置、回転角度)
